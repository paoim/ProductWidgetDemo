Welcome to Product Widget Demo.

It is a demo project that implemented by Spring Boot, AngularJS, HTML, CSS, and Twitter BootStrap. It contains both Backend (Restfull Web Service) and Front-End (Single Page Application).

Backend

  1. Spring Boot
  2. Load Data from CSV files
  3. Support GET and POST HTTP
  4. Support JSON

Front-End

  1. Report - display highest, lowest, and average of products for each employee
  2. Products - display and add New Product
  3. Hours - display and add New Hour
  4. Employees - display and add New Employee
